**Category Menu Sync**
<br>**Menu Items Target**: Choose where the category menu items will be created. You can select the option for displaying the menu items. The drop menu will list all the  positions.
<br>**Menu Item First parent**: Choose a menu parent item under which the top level category menu items will be created. Menu item hierarchy will minimum the categories hierarchy.
<br>**Product category menu**:  Choose settings for creating automatic menu for frontend based on product categories.Disabling sync for any level will disable it for all levels below it irrespective of their setting. You can enable sync with the menu level and you can add the menu level by clicking over the plus icon.![](Screen%20Shot%202020-06-26%20at%204.18.19%20PM.png)
